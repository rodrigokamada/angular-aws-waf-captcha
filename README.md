# Angular AWS WAF Captcha


Application example built with [Angular](https://angular.io/) 21 and adding the [AWS WAF (Web Application Firewall)](https://aws.amazon.com/waf/) CAPTCHA to protect against bots.

This tutorial was posted on my [blog](https://rodrigo.kamada.com.br/blog/adicionando-o-aws-waf-captcha-em-uma-aplicacao-angular) in portuguese and on the [HackerNoon](https://hackernoon.com/how-to-add-the-aws-waf-captcha-to-an-angular-application) in english.
